# hw1-api

Для запуска приложения необходимы java 21 и maven 3.9.х.

Откройте терминал в папке hw1-api, и выполните команды:

```
mvn clean package
```
```
docker compose up
```

Затем импортируйте коллекцию из файла hw1-api/postman/user.postman_collection.json в Postman.

Далее можно запустить коллекцию целиком, или выполнять запросы по отдельности.
