# sn-app

Для запуска приложения необходимы java 21 и maven 3.9.х.

Откройте терминал в папке sn-app, и выполните команды:

```
mvn clean package
```
```
docker compose up
```

Затем импортируйте коллекцию из файла sn-app/postman/user.postman_collection.json в Postman.

Далее можно запустить коллекцию целиком, или выполнять запросы по отдельности.
